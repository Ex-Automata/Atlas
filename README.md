# Atlas — Pathfinder to your code

**Atlas** is a Visual Studio Code extension that helps you **visualize, explore, and understand your codebase**.  
It opens an interactive canvas and displays your files as connected Monaco editors, showing relationships, diffs, and even helping you test code.

---

## ✨ Features

- **Interactive Code Canvas** – View your code in multiple Monaco editors side-by-side.
- **Code Flow Mapping** – See how functions, classes, and modules connect.
- **Diff Visualization** – Quickly compare file versions or branches.
- **Pathfinding** – Trace execution flow and follow dependencies.
- **Testing Hooks** *(planned)* – Run and visualize tests directly from the canvas.

## 🚀 Getting Started

1. **Install** Atlas from the [VS Code Marketplace](#) *(link coming soon)*.
2. **Open Command Palette** → search for **"Atlas: Open"**.
3. Start exploring your code in the interactive canvas.

---

## 🔧 Commands

| Command                 | Description                              |
|-------------------------|------------------------------------------|
| `atlas.open`            | Opens the Atlas canvas                   |
| `atlas.showDiff`        | Displays a diff view between two files   |
| `atlas.traceFlow`       | Highlights code execution flow           |
